### Padding

内边距组件。

```dart
import 'package:flutter/material.dart';

class PaddingExample extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Padding(
      padding: EdgeInsets.only(top: 5.0, right: 10.0, bottom: 5.0, left: 10.0),
      child: Text('Padding')
    );
  }
}
```
